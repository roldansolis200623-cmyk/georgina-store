# Productos

Coloca aquí las imágenes de los productos.

## Tamaños recomendados:
- Imagen principal: 800x800 px (cuadrada)
- Imagen galería: 600x600 px
- Thumbnail: 300x300 px

## Formatos aceptados:
- .jpg / .jpeg
- .png
- .webp (recomendado para mejor rendimiento)

## Estructura sugerida:
```
products/
├── producto-1/
│   ├── main.webp
│   ├── gallery-1.webp
│   ├── gallery-2.webp
│   └── gallery-3.webp
├── producto-2/
│   └── ...
```

## Tips:
- Usa fondo blanco o transparente
- Mantén consistencia en el estilo
- Optimiza las imágenes antes de subir
