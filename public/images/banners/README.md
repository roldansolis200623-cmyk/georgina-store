# Banners

Coloca aquí las imágenes de banners para la página de inicio.

## Tamaños recomendados:
- Banner principal: 1920x600 px
- Banner móvil: 768x400 px
- Banner promocional: 1200x400 px

## Formatos aceptados:
- .jpg / .jpeg
- .png
- .webp (recomendado para mejor rendimiento)

## Nombres sugeridos:
- hero-banner.webp
- promo-banner.webp
- sale-banner.webp
